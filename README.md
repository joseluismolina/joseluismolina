## 🤖 How I Work

> *"I leverage AI as a force multiplier in my development workflow"*

I don't just write code—I architect solutions with AI as my co-pilot. My development approach is built on:

### 📝 Spec-Driven Development
Every project starts with a clear, detailed specification. I use AI to refine requirements, identify edge cases, and create comprehensive technical designs before writing a single line of code.

### 🔧 Agent-Powered Workflows
- **Code Generation**: AI agents for scaffolding, boilerplate reduction, and repetitive tasks
- **Code Review**: Automated review agents catching bugs and suggesting improvements
- **Refactoring**: AI-assisted code modernization and optimization
- **Documentation**: Automated doc generation from code context

### 💬 Prompt Engineering
Crafting precise, contextual prompts for optimal AI outputs. I treat prompt design as a core skill—structuring context, defining constraints, and iterating for quality.

### 🧠 Daily AI Stack
- **GitHub Copilot** for inline suggestions and completions
- **Claude** for deep reasoning and architecture decisions
- **Cursor** for AI-native development environment
- **OpenCode** (open source) for local agent-powered development
- **Ollama** (open source) for running local LLMs (Llama, Mistral, etc.)
- **llama.cpp** (open source) for optimized local inference on CPU/GPU
- **Custom Agents** for project-specific automation
